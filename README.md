# ForRace Motorsport

Site institucional da ForRace Motorsport com foco em apresentacao da equipe, frota, historia, agenda, contato e localizacao da oficina.

## Demo

- Repositorio: `https://github.com/KAUACASANIGA/ForRace.git`

## Funcionalidades

- Landing page responsiva em HTML, CSS e JavaScript
- Secoes de frota, historia, base/cultura e agenda
- Navegacao suave entre secoes
- Efeitos visuais de destaque nos cards ao passar o mouse
- Secao de localizacao com endereco e mapa incorporado
- Links diretos para WhatsApp e Instagram

## Tecnologias

- HTML5
- CSS3
- JavaScript (vanilla)

## Estrutura

- `index.html`: estrutura da pagina
- `css/style.css`: estilos da aplicacao
- `js/app.js`: interacoes e scroll suave
- `img/`: imagens usadas na pagina

## Como rodar localmente

1. Clone o repositorio:
   ```bash
   git clone https://github.com/KAUACASANIGA/ForRace.git
   ```
2. Entre na pasta:
   ```bash
   cd ForRace
   ```
3. Abra o `index.html` no navegador.

## Publicacao

Para publicar via GitHub Pages:

1. Acesse `Settings` -> `Pages` no repositorio.
2. Em `Build and deployment`, selecione:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
